Data Simulasi Pendataan

Bahaya Klik Link Sembarangan dan Mengisi Data pada Link Tidak Dikenal

Halaman ini membantu panitia membuka Google Sheets berisi data demo dari form pendataan warga fiktif dan izin lokasi relawan.

Data relawan harus dihapus setelah seminar selesai. Jangan gunakan form ini untuk mengumpulkan NIK, nomor KK, alamat lengkap, nomor HP, email, password, atau data sensitif lain.

Link akan disimpan di browser panitia ini saja memakai localStorage.

Melihat data terbaru

1. Buka Sheets

Tempel link Google Sheets pada kolom di atas, lalu tekan tombol untuk membukanya.

2. Cek baris terakhir

Data terbaru muncul di baris paling bawah setelah form dikirim dan izin lokasi diberikan.

3. Hapus setelah seminar

Bersihkan isi sheet setelah demo selesai agar data relawan tidak tersimpan lebih lama dari kebutuhan seminar.
